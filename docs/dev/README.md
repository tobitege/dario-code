# Development Artifacts

This directory contains development logs, iteration summaries, status reports, and implementation notes generated during the build process. These are preserved for historical reference but are not part of the active documentation.

## Categories

- **ITERATION_*.md** — Development iteration summaries and findings
- **SANDBOX_*.md** — Sandbox/isolation feature implementation docs
- **SESSIONS*.md** — Session management implementation docs
- **BACKGROUND_TASKS*.md** — Background task system docs
- **TUI_*.md** — Terminal UI implementation status and notes
- **FINAL_*.md / *_STATUS.md / *_COMPLETE.md** — Milestone completion reports
- **EXTRACTION_STATUS.md / REMAINING_CODE_MAP.md** — Code extraction tracking
- **FEATURE_COMPARISON.md** — Feature parity comparison
- **PLUGIN_SYSTEM_CHANGELOG.md** — Plugin system change history
